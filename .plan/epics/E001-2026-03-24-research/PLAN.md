# E001: Research

## Goal
Understand the existing landscape before building anything. Answer:
1. Do similar extensions already exist? What do they do well/poorly?
2. What are the standard cookie categories (GDPR/ePrivacy)?
3. How do popular CMPs (Consent Management Platforms) structure their DOM?
4. Is there an existing open standard for machine-readable cookie consent?
5. What is llms.txt / robots.txt equivalent for AI agents?

## Scope
- Research only — no code in this epic
- Output: comprehensive report in `reports/`
- Findings feed directly into E002 planning

## Tasks
- E001-T01: Research existing cookie consent extensions
- E001-T02: Research cookie categories & GDPR consent standards
- E001-T03: Research popular CMPs and their DOM patterns

## Acceptance Criteria
- [x] Report covers all 5 questions above
- [x] Competitive analysis with feature comparison table
- [x] Cookie category taxonomy documented
- [x] Top 5 CMPs DOM patterns documented (9 done)
- [x] Gaps identified that our project can fill
