Browse freely

Open Standard
Like robots.txt, but for cookie consent
A single JSON file at /.well-known/cookie-consent.json makes your consent UI machine-readable.
{
  "version": "1.0",
  "categories": ["essential", "analytics", "marketing"],
  "cmp": { "name": "cookiebot" },
  "selectors": {
    "acceptAll": "#CybotCookiebotDialogBodyButtonAccept",
    "rejectAll": "#CybotCookiebotDialogBodyButtonDecline"
  }
}
